The Case

The case was to create a website using an api to fetch the data I wanted to be displayed. I went for a cryptocurrency ranking page to show the different cryptos based on market cap.

I ended up using coinranking.com's api to fetch the data I needed. With this I also made a searchbar where you can get any spesific crypto that are listed at coinranking.com up, and using query params to display them on a spesific page.
